title: rate_limit

```yaml
rate_limit:
  resource: ""
```

Throttles the throughput of a pipeline according to a specified
[`rate_limit`](../rate_limits/README.md) resource. Rate limits are
shared across components and therefore apply globally to all processing
pipelines.
